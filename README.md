# 个人博客

> 本项目缺少后端支持，暂时使用 Mockjs 来模拟数据
>
> 未完待续...

## 所用技术

- vue-cli
- vue2
- vuex
- vue-router
- less
- axios

## 项目展望

- 添加登录界面，使用 vue-router 的路由导航守卫实现登录鉴权
- 后端打算使用 node 搭建轻量级服务器，为前端提供 RESTFUL 风格的 api
- 使用 mysql 做数据持久化处理
